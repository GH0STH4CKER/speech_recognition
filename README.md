# Python Speech Recognition


<h2>Installing Required Modules</h2>
```pip install SpeechRecognition```
```pip install PyAudio```
</br>
If error occur when installing pyaudio :
*Windows 
```pip install pipwin```
```pipwin install PyAudio```
*Linux
```apt-get install python-pyaudio```
